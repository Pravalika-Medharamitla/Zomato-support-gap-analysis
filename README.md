# Zomato Customer Support Gap Analysis

**PM Portfolio Project | Pravalika Medharamitla | May 2026**

## Overview

Analysed 50 negative user reviews of Zomato to identify recurring 
customer support issues and recommend data-backed product improvements.


## Problem Statement

Zomato users frequently report unresolved complaints around delayed 
deliveries, payment failures, and poor support experience. This project 
identifies the most critical pain points and prioritises solutions 
using the RICE framework.

## Methodology

- 50 user reviews manually collected and categorised into 5 problem types
- Severity rated as High / Medium / Low for each review
- COUNTIF analysis in Google Sheets used to quantify patterns
- Top 3 issues prioritised using RICE framework

## Key Finding

58% of all complaints fall under just 2 categories:
- Order & Delivery — 40%
- Payment & Refund — 18%


## RICE Prioritisation
- **Payment & Refund** — RICE Score: 180 ⭐ Highest Priority
- **Order & Delivery** — RICE Score: 145.8
- **Poor Support** — RICE Score: 74.7

## Top Recommendations

1. **Automated Refund Engine** (RICE: 180) — Auto-process refunds within 24 hours
2. **Proactive Delay Notifications** (RICE: 145.8) — Push alerts with revised ETA
3. **In-App Live Chat** (RICE: 74.7) — Human agent access for serious complaints
